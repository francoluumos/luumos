# Lovable App

Hier wird der Lovable-generierte Frontend-Code synchronisiert, sobald das Lovable-Projekt mit GitHub verbunden wird.

**Status:** Noch leer. Lovable-Projekt noch nicht angelegt.

## Setup-Schritte (wenn so weit)

1. Lovable-Projekt anlegen ([lovable.dev](https://lovable.dev))
2. Im Lovable-Projekt: GitHub-Integration aktivieren, dieses Repo wählen, Pfad = `lovable-app/`
3. Initiales Lovable-Bauen mit Brand-Tokens aus `design-system/tokens.css` einbinden
4. Supabase-Connection einrichten (URL und Anon-Key in Lovable Environment Variables)
5. Edge Functions in separatem Pfad (`/supabase/functions/`) — Supabase CLI deployt diese
